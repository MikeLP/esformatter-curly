# `ImportDeclaration`

## original

```js
import { x, y, z } from 'utils';

import {
  x,
  y,
  z
} from 'utils';
```



## format

```js
import { x, y, z } from 'utils';

import { x, y, z} from 'utils';
```



## format with esformatter-curly

```js
import { x, y, z } from 'utils';

import {
  x,
  y,
  z
} from 'utils';
```
